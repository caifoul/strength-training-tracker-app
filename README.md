# Strength Tracker App Template

A simple client-side strength training tracker built with HTML, CSS, and JavaScript.

## Features

- Log exercises with sets, reps, and weight
- View workout history
- Persist data in `localStorage`
- Clear workout log

## Getting Started

1. Open `index.html` in your browser.
2. Add exercises using the form.
3. Your workout log is saved automatically.

## Extend the Template

You can extend this starter app by adding:

- exercise categories or muscle groups
- workout sessions and dates
- performance charts and progress tracking
- edit/delete controls for logged exercises
